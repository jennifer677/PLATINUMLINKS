# Platinum Custom Co. Editable GitHub Files

Upload these files directly to the root of your GitHub repo:
- `index.html`
- `style.css`
- `script.js`
- `config.js`
- `logo.png`
- `README.md`

## Easy edits
Edit `config.js` first whenever you want to change:
- brand text
- link URLs
- button labels
- service lists

## Notes
- Keep the logo file named `logo.png`.
- Publish GitHub Pages from the root of the repo.
